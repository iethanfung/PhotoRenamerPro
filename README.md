# Photo Renamer Pro

专业的照片重命名和分类工具，专为Apple新产品可靠性测试设计。

## 功能特点

- 智能解析手工命名的照片文件
- 动态配置命名模板和文件夹结构
- 自学习系统，越用越聪明
- 支持标准照和问题照分类处理
- macOS Sonoma风格界面

## 安装

1. 安装Python 3.11+
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
## 启动项目
   ```bash
     python main.py
   ```
    
## 使用说明
1. 配置Excel文件路径
2. 设置输出文件夹
3. 拖入照片文件或选择文件夹
4. 预览解析结果并进行必要修正
5. 开始处理
